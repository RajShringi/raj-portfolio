# Why I Prefer Building Projects Over Watching Tutorials

Most of my learning has come from building projects rather than endlessly watching tutorials.

When I start a project, I naturally run into problems:

- How should I structure components?
- Why is state updating incorrectly?
- How do I make the UI responsive?
- Why is this API request failing?

Those moments force me to think, search, experiment, and understand concepts more deeply.

Tutorials are useful for getting started, but real learning happens when you try to build something on your own and get stuck.

Recently, I’ve been focusing more on:

- Building small projects
- Improving UI design
- Writing cleaner React code
- Learning by solving real problems

Even small projects teach a lot when you build them consistently.
